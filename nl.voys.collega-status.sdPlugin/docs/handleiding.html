Voys Collega Status

Stream Deck-plugin — handleiding voor installatie en gebruik

Wat doet deze plugin?

Met Voys Collega Status zie je op je Elgato Stream Deck de beschikbaarheid van collega's uit je Voys-omgeving. Elke knop toont één collega met naam, status en optioneel het interne nummer.

Statussen komen live binnen via de Voys Resgate-websocket — dezelfde bron die de officiële Voys-app en webphone gebruiken voor collega-beschikbaarheid.

Vereisten

  • Elgato Stream Deck versie 6.5 of nieuwer (Windows 10+ of macOS 10.15+)
  • Node.js — wordt automatisch gebruikt door de plugin; installeer van nodejs.org als de plugin niet start
  • Een Voys-account met een geldig API-token
  • Netwerktoegang tot api.voys.nl en resgate.eu-production.holodeck.voys.nl

Installatie

  1. Download het bestand voys-collega-status.streamDeckPlugin.
  2. Dubbelklik het bestand — Stream Deck opent en installeert de plugin.
  3. Sleep de actie Collega Status naar een vrije knop op je Stream Deck.
  4. Klik op de knop om het instellingenvenster (Property Inspector) te openen.

Bij de eerste start installeert de plugin automatisch zijn Node.js-dependencies (ws). Dat kan enkele seconden duren.

Configuratie — verbinding

Stel eerst de globale verbinding in onder Verbinding:

1. API-token invullen

Vul je persoonlijke Voys API-token in. Dit token koppel je aan je Voys-account en geeft toegang tot collega's en statusinformatie binnen jouw organisatie.

Deel je API-token nooit publiek. Bewaar het alleen in Stream Deck via Opslaan. Het token wordt lokaal opgeslagen in de Stream Deck-instellingen van jouw computer.

2. Client ID detecteren

Klik op Detecteer client_id. De plugin haalt automatisch je organisatie-ID en client UUID op. Controleer of de velden Client UUID en Client ID zijn ingevuld.

3. Verbinding testen

Klik op Test verbinding. Bij succes zie je Token geldig. Klik daarna op Opslaan om de instellingen te bewaren.

Geavanceerde velden (meestal niet nodig)

  • Voys API URL — standaard https://api.voys.nl/api/v2
  • Resgate URL — standaard wss://resgate.eu-production.holodeck.voys.nl
  • Click-to-Dial URL — standaard https://api.voipgrid.nl/api/clicktodial

Pas deze alleen aan als Voys andere endpoints heeft aangegeven voor jouw omgeving.

Configuratie — collega per knop

Elke Stream Deck-knop kan één collega tonen. Herhaal deze stappen per knop:

  1. Klik op Collega's laden om de lijst op te halen.
  2. Kies een collega uit het dropdown-menu.
  3. Pas optioneel de weergave aan: naam, status en/of intern nummer.
  4. Klik op Opslaan voor deze knop.

Druk op een geconfigureerde knop om de collega te bellen (via je webphone/app) en de status te verversen. Zorg dat de collega een intern nummer heeft.

Collega bellen

Bij indrukken van een geconfigureerde Collega Status-knop start de plugin een uitgaand gesprek via de Voys Klik-en-bel API. Eerst gaat je eigen eindbestemming over (webphone, app of toestel — zoals ingesteld in het Voys-portaal), daarna wordt het interne nummer van de collega gebeld.

Vereisten

  • De collega heeft een intern nummer (zichtbaar na Collega's laden).
  • Je hebt een Klik-en-bel/eindbestemming ingesteld in Voys (zelfde instelling als de browserextensie).
  • Je webphone of app is actief als je via webphone/app belt.
  • Partneraccounts hebben geen toegang tot Klik-en-bel — alleen normale gebruikers.

De plugin gebruikt hetzelfde persoonlijke API-token als voor status; authenticatie voor bellen verloopt via Token e-mail:token naar api.voipgrid.nl.

Statussen

De knop toont één van de volgende statussen:

Vrij
Intern
Bezet
Offline
Onbekend
Auth
Config
  • Vrij — collega is bereikbaar
  • Intern — alleen bereikbaar voor collega's
  • Bezet — niet storen, in gesprek of aan het bellen
  • Offline — niet ingelogd of geen actief toestel
  • Config — knop nog niet geconfigureerd (geen collega gekozen)
  • Auth — token ongeldig of geen toegang tot statusdienst

Problemen oplossen

Knop toont "Config"

Selecteer een collega en klik op Opslaan voor deze knop.

Knop toont "Auth"

  • Controleer of het API-token nog geldig is in het Voys-portaal.
  • Klik op Test verbinding en daarna Opslaan.
  • Controleer of Client UUID en Client ID correct zijn (via Detecteer client_id).

Collega's laden mislukt

  • Token testen en opslaan.
  • Client ID handmatig invullen als detectie faalt (te vinden in het Voys-partnerportaal).

Status verandert niet

  • Druk op de knop om te verversen.
  • Herstart Stream Deck (plugin maakt automatisch opnieuw verbinding).
  • Controleer of de Resgate URL ongewijzigd is gelaten.

Bellen mislukt (fouttekst op knop)

  • Vul je E-mail (Klik-en-bel) in onder Verbinding en klik Opslaan.
  • Controleer je Klik-en-bel-instelling in het Voys-portaal (Telefooninstellingen).
  • Open je webphone of app vóór je op de knop drukt.
  • Partneraccounts kunnen Klik-en-bel niet gebruiken.
  • Logbestand: plugin.log in de plugin-map (map op de knop → Open in Explorer).

Knop toont "Geen nr"

De gekozen collega heeft geen intern nummer. Kies een andere collega of vraag een beheerder om het interne nummer in Voys in te stellen.

Plugin start niet

  • Installeer Node.js (LTS-versie).
  • Herinstalleer de plugin via het .streamDeckPlugin-bestand.

Privacy

De plugin slaat je API-token en instellingen lokaal op via Stream Deck. Alle communicatie verloopt rechtstreeks tussen jouw computer en de Voys-API's — er worden geen gegevens naar externe partijen gestuurd.
